\# WorkspaceWiz



WorkspaceWiz is a smart workspace booking platform designed to simplify office space reservations.



\## Features



\- Email OTP Authentication

\- Workspace Search and Filtering

\- Seat Booking

\- QR Code Based Booking Confirmation

\- Wishlist Management

\- Admin Dashboard

\- Workspace Analytics



\## Tech Stack



\### Frontend

\- React

\- TypeScript

\- Lovable



\### Backend

\- Flask

\- MongoDB

\- JavaScript

\- HTML

\- CSS
